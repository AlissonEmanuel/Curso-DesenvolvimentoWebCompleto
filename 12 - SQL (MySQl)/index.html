DML - Data Manipulation Language - Linguagem de manipulação de dados

DCL - Data Control Language - Linguagem de controle de dados

DTL - Data Transaction Language - Linguagem de transação de dados

DQL - Data Query Language - Linguagem de consulta de dados

Tipos de dados SQL (MySQL)

Campos de texto:

  • Text (tamanho variável que armazena uma grande quantidade de caracteres)
  • Varchar (tamanho variável que armazena de 0 até 255 caracteres)
  • Char (tamanho fixo que armazena de 0 até 255 caracteres)

Campos numéricos:

  • Int (valores númericos inteiros)
  • Float (valores númericos fracionados)

Campos de data e hora:

  • Date (data no formato yyyy/mm/dd)
  • Time (hora)
  • Datetime (combinação de date e time em um mesmo campo)

Incluindo, editando e removendo colunas de tabelas

CREATE TABLE table_name (column1 datatype, column2 datatype, column3 datatype, .... );

ALTER TABLE table_name ADD column_name datatype;

ALTER TABLE table_name DROP COLUMN column_name;

ALTER TABLE table_name ALTER COLUMN column_name datatype;

Inserindo dados na tabela

INSERT INTO table_name (column1, column2, column3, ...) VALUES (value1, value2, value3, ...);

Consultado dados

SELECT * FROM tb_cursos

SELECT * FROM tb_cursos where nome_curso = 'Curso de Node.JS com MongoDB'

Filtrando tabelas com operadores

Operadores de comparação

  • = valor da esquerda igual o valor da direita
  • < valor da esquerda menor que o valor da direita
  • <= valor da esquerda menor ou igual ao valor da direita
  • > valor da esquerda maior que o valor da direita
  • >= valor da esquerda maior ou igual ao valor da direita

Operadores lógicos

  • AND - Todas as operações de comparação devem ser verdadeiras
  • OR - Pelo menos uma das comparações devem ser verdadeiras

Outros operadores

  • BETWEEN - Seleciona valores de acordo com a instrução fornecida, os valores podem ser números, textos ou dados. Por ex: SELECT investimento FROM tb_cursos WHERE investimento BETWEEN 500 and 700;
  • IN - Permite especificar múltiplos valores em um WHERE. Por ex: SELECT interesse FROM tb_alunos WHERE interesse IN('Jogos', 'Música', 'Esporte');
  • LIKE - Permite realizar filtros com base em uma pesquisa de caracteres dentro de uma coluna textual. Caracteres curingas: % Indica que pode haver a existência de qualquer conjunto de caracteres no texto. _ Indica que pode haver a existência de um ou mais caracteres em uma posição específica do texto. Por ex: SELECT nome FROM tb_alunos WHERE nome LIKE '%tt_';

Ordenando resultado

ASC ou DESC - ASC ordena de forma ascendente de DESC de forma descendente

Limitando retorno

Utiliza-se LIMIT e OFFSET para limitar o retorno da query sql, LIMIT sendo a quantidade de registros a serem exibidos e OFFSET o início dos registros que serão exibidos. Por ex: SELECT nome FROM tb_alunos LIMIT 5 OFFSET 2. Também pode ser utilizado a escrita LIMIT 2, 5 por exemplo.
